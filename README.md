# krishi-kshetra-website
Official website for Krishi Kshetra — an agricultural robotics and smart irrigation startup focused on sustainable farming automation.

## Overview

This repository contains the official website source code for Krishi Kshetra, an agricultural robotics and smart irrigation startup focused on sustainable farming automation and intelligent agricultural technology.

The website is designed to showcase:
- Startup vision
- Smart irrigation solutions
- Agricultural automation systems
- Technology roadmap
- Team and project information
